This is a simple expansion board used to add both thermistor measurement and i2c expansion, mainly for off-board temperature and humidity sensors like SHTxxx devices. This PCB is also designed in [KiCad](https://www.kicad.org/). Fabrication and is again tuned toward [JCLPCB](https://jlcpcb.com/). 100% hand-soldered.

* **Rev A**: Has been fabricated, assembled, tested and is use in four places. No issues.
